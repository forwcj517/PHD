Installation

Preparing Installation

  1. Download ZIP package and unzip it.
  2. In unzipped folder you will find folder directory like bellow.
    • phd
    • database
    • documentation
  3. Create app directory on your web directory server (Xampp/htdocs/phd)
  4. Simply put all files inside directory phd to your app folder.

Create new database installation

You need to create new database named "prohealthdetect_sensor_info" , you can use phpmyadmin or any mysql databse tools such sqlyog etc
---
After new database created , now import database/prohealthdetect_sensor_info.sql file into your new database

Configuration Installation

1. Open phd/.env file using your favorite php editor

2. Setting Debug mode and App Url

	APP_DEBUG=false
	APP_URL=http://localhost/phd

3. Setting Database Info

Open .env file using your favorite php editor 

	DB_CONNECTION=mysql
	DB_HOST=127.0.0.1
	DB_PORT=3306
	DB_DATABASE=prohealthdetect_sensor_info
	DB_USERNAME=root
	DB_PASSWORD=

4. Access your application

  1. Now access your application via your favorite browser
    http://127.0.0.1/phd
  2. Default Login :
    Email Address : admin@admin.com
    Password : admin
  3. Awesome your application is ready now !
